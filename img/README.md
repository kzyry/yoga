# Иконки авторов

## Требования к изображениям

- **Размер:** 24x24px (минимум) или кратный размер (48x48px, 96x96px для Retina)
- **Формат:** JPG, PNG, WebP
- **Название:** точное имя автора из JSON (например, `darin.yoga.jpg`)
- **Вес:** до 50KB для оптимальной загрузки

## Структура файлов

```
img/
├── darin.yoga.jpg
├── another.author.jpg
├── example.user.png
└── README.md
```

## Как добавить иконку

1. Найдите имя автора в таблице (колонка "Автор")
2. Сохраните аватар автора с именем `{author}.jpg` в эту папку
3. Обновите страницу - иконка появится автоматически

## Примечания

- Если изображение не найдено, оно просто не отобразится (без ошибок)
- Изображения автоматически обрезаются до круглой формы
- Рекомендуется использовать квадратные изображения для лучшего отображения

## Массовое добавление

Для добавления всех аватаров авторов можно использовать скрипт:

```bash
# Получить список уникальных авторов
cat filtered_50k_views.json | jq -r '.[].author' | sort -u > authors.txt

# Далее скачать аватары для каждого автора
```
